# 🚀 To-Do React Avançado

![Tela de aplicativo de lista de tarefas mostrando tarefas pendentes e concluídas, com campos para adicionar novas tarefas e interface organizada de React](src/assets/todo-avancado.png)
(Imagem disponível no arquivo README.md dentro do projeto).

Este projeto é uma aplicação de lista de tarefas (Todo List) desenvolvida em React com foco em:

- Hooks nativos do React
- Hooks customizados
- Memoization com `useMemo` e `React.memo`
- Context API para estado global
- Persistência em `localStorage`

## 🛠️ Tecnologias Utilizadas

- **React** v19
- **Vite**
- **JavaScript**
- **CSS**
- **Context API**
- **Custom Hook** `useLocalStorage`

## 🚀 Scripts disponíveis

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## 🏁 Como rodar o projeto localmente

1. **Clone o repositório:**
   ```bash
   git clone <repo-url>
   ```

2. **Entre na pasta do projeto:**
   ```bash
   cd my-app
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. Abra o navegador em `http://localhost:5173`.
